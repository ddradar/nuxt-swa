## Install

1. Add the module to your devDependencies

::code-group
```bash [pnpm]
pnpm install -D nuxt-swa
```

```bash [yarn]
yarn add -D nuxt-swa
```

```bash [npm]
npm install --save-dev nuxt-swa
```
::

2. Add module to `nuxt.config.ts`

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['nuxt-swa'],
  swa: {
    /* Your config here (optional) */
  },
})
```
