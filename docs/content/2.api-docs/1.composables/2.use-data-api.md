Returns pre-configured `$fetch` object for Data API.

::tip
Data API feature on Azure Static Web Apps is **public preview**.
::

## Return Values

### `$graphql`

`$fetch` object pre-configured for GraphQL API

### `$rest`

`$fetch` object pre-configured for REST API
