::warning
**Deprecated**: Azure Static Web Apps has discontinued Data API support. This composable and the related `swa.dataApi` option will be removed in a future version.
::

Returns pre-configured `$fetch` object for Data API.

## Return Values

### `$graphql`

`$fetch` object pre-configured for GraphQL API

### `$rest`

`$fetch` object pre-configured for REST API
